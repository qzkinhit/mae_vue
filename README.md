# README.md

## 一些说明

本文件夹为基于Vue框架的“见微”系统-员工打卡场景开发源码，开发基于vue-element-admin模板，其中人脸检测模块调用tracking.js，依赖包管理采用npm，运行前需通过`npm install`安装依赖库，请各位老师审阅。

## 几个重要的项目目录

-- node_modules		项目依赖包

-- src

​		-- api 						后端请求接口

​		-- componets			前端组件

​		-- router				   导航栏路由管理

​	 	-- views					页面主文件(.vue) 

​				-- auth.vue		认证页面

​				-- login/index.vue				登录页面

​				-- dashboard/index.vue		员工演示主页
